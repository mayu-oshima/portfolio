制作ポイント

React + TypeScriptでファッション通販サイトを制作しました。以下の点を特に工夫しています。

商品ページのルーティング設計

React Router を用いて、商品一覧・詳細・カート・注文完了ページへの遷移を実装。
URLパラメータを活用して、商品IDから動的に詳細情報を表示できる構成にしています。

カート機能のグローバル管理

React Context を使ってカートの状態（商品情報や数量など）を全体で共有。
追加・削除・リセットなどのロジックをContext内にまとめて、各ページから操作できるようにしました。

フォームと注文完了ページの導線

氏名・住所などの入力フォームを作成し、送信後は「注文ありがとうございます」ページへ遷移。
注文完了時にはカートの中身を自動でリセットし、UXにも配慮しています。

styled-componentsによるUI設計

styled-componentsで各パーツをコンポーネント単位に分離。
配色やタイポグラフィ、余白などを整え、ECサイトらしい清潔感ある見た目に仕上げました。
